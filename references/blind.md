# §4 BLIND — 맹점 감지 엔진

**목적:** 가설 내부에서 **주체가 못 보는 것**을 흔적·부재·비대칭 3신호로 역추적. 맹점을 "직접 찾는다"는 정의상 모순 → **간접 증거만**.

**트리거 연동:**
- **홈즈** (①관찰 ②이상징후 ③복수가설 ④증거대조 ⑤소거) — S1·S3 감지 엔진
- **절대자** (형 프레임 고정 + 빠진부분 전수탐색 + MECE) — S2 빈자리 감지
- **오컴** (가정최소 채택) — 원인 분류 판정

**설계 철학:** 발견 = 보너스, 숨김 = 페널티 (→ `rationale.md` R1 넛지 적용).

> [!info] 모드 분기
> - **LIGHT:** S1·S3만 (축당 1줄 판정). S2는 ACH 미가동이라 면제.
> - **DEEP:** S1·S2·S3 전 신호 + 감지 시 §5 연쇄 트리거
> - **TURBO:** 축별 Agent 병렬 실행

---

## 4-1. 3신호 체계 (S1·S2·S3)

| 신호 | 정의 | 트리거 조건 | 감지 소스 |
|------|------|-----------|----------|
| **S1 비대칭** | 축 간 증거밀도 극단 불균형 | 한 축 증거수 ÷ 평균 < 0.3 또는 > 3.0 | §2-1 팩트 장부 |
| **S2 부재** | ACH 매트릭스 구조적 공백 | 2개 이상 가설에서 빈칸(?) | §2-3 ACH |
| **S3 반증 공백** | 반대증거 탐색 결과 0건 | §2-2 반대증거 목록 = 0 | §2-2 허브 |

---

## 4-2. S1 비대칭 — 홈즈 관찰

**가설:** 3축 독립조사인데 한 축만 극단이면 **축 정의 오염** or **진짜 축이 다른 곳**.

**계산:**
```
평균 증거수 = 전체 팩트 장부 ÷ 축 수
축_i < 평균 × 0.3 → 과소 (진짜 축 아닐 수 있음)
축_i > 평균 × 3.0 → 과다 (다른 축 흡수 가능성)
```

**선험적 비대칭 예외 (§1-3 프레임 선언 시점만, 사후 무효):**
- 규제·법령 축 (법령 1건이 상용 축 상회)
- 핵심인물 축 (단일 결정력 > 시장)
- 독점 기술 축 (특허 1건이 산업 재편)

선언 형식: `[축명]: 선험적 비대칭 면제 — 사유: [규제/인물/기술/기타]`

**감지 시 조치 (오컴: 최소 가정):**

| 과다/과소 | 해석 | 액션 |
|-----------|------|----------|
| 과다 | 축 정의 오염 | §2-4 프레임 수준 회귀 고려 |
| 과소 | MECE 오설계 | §1-3 재검증 |
| 동시 | 프레임 재설계 | §5 T1 자동 발급 |

---

## 4-3. S2 부재 — 절대자 빈자리

**가설:** ACH 빈칸(?, 평가불가) 2개+ = **가설 공간 불완전**. N(Neutral)은 빈칸 아님 (적극 중립 판정).

**감지:** §2-3 ACH 작성 후 빈칸 수 ≥ 2.

**원인 분류:**
- 증거 부족 → §2-1 재조사 (§2-4 가설 수준 회귀)
- 가설 누락 → §1-2 경쟁 가설 추가 → §5 T1 연쇄
- 판단 불가 → 가설 누락으로 기본 처리(보수적)

---

## 4-4. S3 반증 공백 — 홈즈 소거

**가설:** 반대증거 0건 = **반증불가 가설(비과학적)** or **탐색 불성실**.

**감지:** §2-2 허브 완료 시 반대증거 = 0.

**이중 확인 (자기기만 방지):** S3 발동 시 **자동 1회** 재탐색 허용 (§2-2 단일지점 예외, INV 7).
- 쿼리: "이 가설이 틀렸다는 관점·반대 사례·실패 사례"
- 여전히 0건 → **반증불가 가설** 확정
- 발견 → §2-2 목록 추가

**조치 (확정 후):**

| 원인 | 조치 |
|------|------|
| 반증불가(동어반복·너무 일반적) | §1-2 재설계 |
| 너무 극단적 | §5 T2 자동 발급 (중간 가설 탐색) |
