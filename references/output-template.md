# 산출물 내부 구조 템플릿

축별 파일(`_research/[축명].md` 또는 `_research/LIGHT_[축명].md`) 안에 들어갈 섹션 순서와 필수 항목. 피라미드 원칙(결론 선행) + 인텔리전스 브리핑(증거 계층화) + PRISMA(투명성)를 반영.

---

## DEEP 템플릿 — `_research/[축명].md`

```markdown
# [축명]: [핵심 주제]

## 핵심 판정
**[결론 2~3문장 서술.]** 이 판정이 [의사결정]에 [어떻게] 영향하는지까지 포함.
1차 출처 1개 이상 인라인 인용 필수. (출처: [URL], YYYY-MM)

## 배경 및 맥락
[이 축을 조사하게 된 이유, 전제 가정, 조사 범위를 2~3문장으로 서술.]

## 분석
[발견한 것이 무엇인지, 왜 중요한지, 어떻게 연결되는지 — 3~5문단 서술 의무.
단순 나열 금지. 각 문단은 하나의 논점을 전개하고 1차 출처를 인라인 인용.
예: "X에 따르면 (출처: URL, YYYY-MM), … 이는 Y를 의미한다. 반면 Z는 …"]

## 반대 근거
[주요 반론 또는 반증을 2~3문장 서술. "만약 이것이 참이면 → [시나리오]" 포함.
판정: 기각 / 경쟁 가설 편입 / 조건 분기]

## 결론 및 시사점
[핵심 판정을 재확인하고, 의사결정에 직접 연결되는 실행 함의를 2~3문장으로 서술.]

## 한계
[접근 불가 소스, 시간 제약, 불확실성 — 판정에 미치는 영향(치명적/경미) 포함.]
```

## EXPAND 템플릿 — `_research/EXPAND_[T1|T2|T3]_[축명].md`

```markdown

## 복귀 서약 (타임스톤)
- 토큰: [T1 틀밖 / T2 아날로지 / T3 줌]

## 탐색 경과
```

## 규칙

- **서술 본문 의무 (v2.8):** 분석 섹션 3~5문단 서술 필수. 표만 있고 본문 없음 = FAIL
- **결론 선행 필수:** "핵심 판정" 섹션이 파일 최상단. 서술 2~3문장 + 출처 인용 포함
- **표는 부록으로만 (v2.8):** 팩트장부·5장르·수렴4축·맹점 = 검증 부록 섹션에만. 본문에 표 삽입 = FAIL
- **인라인 인용 필수 (v2.8):** 분석 본문 각 문단에 (출처: URL, YYYY-MM) 인라인 인용. 없으면 미검증
- **대안분석 필수 (DEEP):** "반대 근거" 섹션 없는 DEEP 산출물 = 미완성
- **맹점 로그 필수 (DEEP, INV 7):** 감지 여부와 무관하게 S1·S2·S3 판정 기록
- **LIGHT는 경량 유지:** 3섹션(판정·근거·주의사항) 초과 금지

---

- **본문 인용 필수 (v2.7, INV 10):** 결론 한 줄당 1차 출처 1+ 인용. 표·등급만 있고 본문 인용 ✗ = FAIL
- **5장르 대비표 필수 (v2.7, INV 11):** 단일 장르 결론 = FAIL. 5장르 발견 대비표 미작성 = FAIL
- **근거ID 매핑 필수 (v2.7, INV 12):** 별점·강중약·우열 등 평가 옆 근거ID 누락 = FAIL
- **양적 포화 N+ 필수 (v2.7, B1):** 깊이별 N 미달 시 "추정" 어조 강제. 단정 = FAIL
- **출처 시점 필수 (v2.7, B5):** 모든 출처에 (YYYY-MM) 명시. 누락 = FIG 미통과
- **1차 출처 라벨 필수 (v2.7, B2):** 모든 팩트에 1차/2차/3차 라벨. 3차 단독 결론 = FAIL

## 경량화 규칙 (분량 상한)

분량 팽창은 판정 가독성을 훼손하고 의사결정 지연을 유발한다. 각 섹션에 상한을 두고, 초과분은 진단성·영향도 기준으로 선별하여 절사한다.

| 섹션 | 상한 | 초과 시 처리 |
|------|------|-------------|
| 분석 문단 | **5문단** | 논점 기여도 상위 5개만. 나머지 각주로 |
| 팩트 장부 (FIG) | **장르당 3개** | 영향도·확실성 상위 3개 |
| 반대 근거 | 핵심 가설당 **1-2개** | 시나리오 설득력 높은 것 우선, 나머지는 각주로 |
| LIGHT 근거 3개 | **고정 3개** | 초과 시 DEEP으로 모드 전환 고려 |

**원칙:** 상한은 하한이 아니다 — 진단성·영향도가 낮으면 상한 미달도 허용. 단, LIGHT 근거 3개는 고정.
