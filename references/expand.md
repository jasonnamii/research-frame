# §5 EXPAND — 프레임밖 탐색 엔진

**목적:** 현재 프레임의 **구조적 한계 밖**을 탐색. 맹점의 70%는 프레임 밖.

**트리거 연동:**
- **틀밖** (프레임 박제·거부·3대 킬샷) — T1 대안프레임 엔진
- **아날로지** + **줌** — T2·T3 범주·시공간 이탈 엔진
- **트리아지** — 셸링포인트·복귀 판정
- **타임스톤** — 복귀 서약·턴 타이머

**설계 철학:** 예산제 + 객관 복귀 + 셸링포인트 (→ `rationale.md` R2·R3·R4).

> [!warning] 모드 분기
> - **LIGHT:** **§5 전체 금지** — 폭 유지·깊이 최소
> - **DEEP:** T1·T2·T3 토큰 예산제
> - **TURBO:** 축별 Agent 병렬 탐색

## 5-1. 토큰 예산 (동적)

축 수 × 0.5 (반올림 올림).

| 축 수 | Token |
|------|------|
| 2축 | 1 |
| 3~4축 | 2 |
| 5축+ | 3 (상한) |

토큰 종류 3종(T1·T2·T3). 총 예산 내 자유 선택. 같은 종류 중복 가능.

---

## 5-2. T1 — 대안 프레임 (틀밖 발동)

**정의:** "X가 아니라 Z라면" — 현재 프레임 전제 뒤집기.

**절차:**
1. 근본 가정 1줄: "우리는 [X]라고 전제하고 [Y] 축으로 본다"
2. **틀밖** 3대 킬샷 발동:
   - (a) 진짜 답이 "이걸 안 하는 것/안 쓰는 것" 아닌가
   - (b) 대상·구조·방식 자체를 바꿔야 하지 않나
   - (c) 아무도 안 묻는 질문은?
3. 대안 프레임으로 축 2개+ 도출 + 증거 2개+
4. 복귀 판정(5-5)

**발동:** 형 명시 / §4 S1 과다+과소 동시 / §4 S2 가설 누락 / §2-4 프레임 회귀. **토큰 비용: 1**

---

## 5-3. T2 — 범주 이탈 (아날로지+줌)

**정의:** 상위 카테고리 이탈. 마케팅 → 조직문화, 기술 → 규제.

**T1과의 차이:** T1=같은 카테고리 내 대안. T2=카테고리 자체 이탈.

**절차:**
1. 카테고리 식별: "이 리서치는 [카테고리] 문제"
2. **아날로지**(구조적 패턴 추출 + 타도메인 매핑)로 인접 카테고리 3개 도출
3. 설득력 최상위 1개 선택
4. 해당 카테고리로 축 2개+ 재설계 + 증거 2개+
5. 복귀 판정

**발동:** 형 명시 / §4 S3 가설 극단적 / 교차분석 축간 모순 해소 불가. **토큰 비용: 1**

---

## 5-4. T3 — 시공간 이탈 (줌+아날로지)

**정의:** 시점·지역·스케일 급변. 2026 한국 → 1995 일본, 개인 → 산업, 1년 → 10년.

**절차:**
1. 현재 좌표 명시: "현재 [시간]·[공간]·[스케일]"
2. **줌** (미시↔거시 스케일 선택 + 개인/팀/조직/산업/사회 재조명) 발동
3. 이탈 좌표에서 **아날로지** 패턴 수집
4. 현재 맥락 적용 가능성 판정
5. 복귀 판정

**발동:** 형 명시 / 확산 5유형 "인접산업·실패패턴"에서 이탈 필요 시. **토큰 비용: 1**

---

## 5-5. 복귀 조건 (트리아지 발동)

**객관 메트릭만.** Claude 주관 금지. → `rationale.md` R2

**메트릭 1 · 턴 타이머:** 토큰 사용부터 **최대 3 세션 턴**. 초과 = 자동 복귀 강제.

**메트릭 2 · 증거 충족:** 수렴4축 중 **2축 충족**하면 성공 복귀.

| 턴 수 | 수렴4축 | 판정 |
|-------|---------|------|
| ≤3턴 | 2축+ | 성공 — 편입 |
| ≤3턴 | 1축 이하 | 실패 — 기록만 |
| 3턴 초과 | 무관 | 강제 — 기록만 |

**금지:** "조금만 더" "감이 왔어" "거의 다 왔어" = 주관 연장.

## 5-7. 복귀 서약 (타임스톤 발동)

**토큰 사용 전 필수 4항 선언:**

```
[EXPAND 토큰 사용 선언]
- 토큰: [T1/T2/T3]
- 사유: [발동 조건]
- 가설: [대안 프레임/범주/시공간]
- 복귀 조건: 3턴 내 수렴4축 2축+ = 성공, 아니면 실패
- 주가설 동결: [상태 기록]
```

**위반:**
- 3턴 초과 자의 연장 → 해당 축 신뢰도 2tier↓
- 서약 없이 토큰 사용 → 토큰 미사용 처리 + 결과 무효
