# §1 WHY — 왜 조사하는가

리서치의 존재 이유 정의. 이 층위 없으면 리서치가 자기목적화.

**트리거 연동:**
- **제1원리** (유추·관행 차단 + 환원불가 자명명제 + 재조립) — 의사결정 연결점 도출
- **백본** (결론먼저·역방향·필수논거만) — 가설 재진술
- **틀밖** + **절대자** — MECE 검증 자기반박

> [!info] LIGHT 모드
> - 1-1: 1줄 추정 (매핑 생략)
> - 1-2: 1줄 가설만 (3조건 점검 생략)
> - 1-3: 축은 DEEP과 동일, 전제 점검·형 승인 생략
> - WHY 게이트: 면제 — 바로 §2
>
> LIGHT→DEEP 전환 시: LIGHT의 1줄 추정·가설·축이 DEEP WHY 1-1/1-2/1-3의 초안.

## 1-1. 의사결정 연결점 (제1원리 발동)

"이 리서치 결론이 어떤 의사결정을 바꾸는가?"

**제1원리** 발동: 유추·관행 차단 → 환원불가 자명명제까지 분해 → 의사결정으로 재조립.

**절차:**
1. 의사결정 1-3개 명시 (형이 안 줬으면 추정 + 확신도 태깅)
2. 결론→선택지 매핑: "결론 A면 X, 결론 B면 Y"
3. 매핑 불가 주제 = 스코프 밖 후보

**예시:**
```
의사결정: "이 시장에 진입할 것인가?"
  → 시장규모 ≥ X + 경쟁강도 ≤ Y → 진입
  → 그 외 → 보류 또는 피벗
```

## 1-2. 가설 수립 + 3조건 (백본 발동)

**백본** 발동: 결론먼저 → 역방향 전개 → 필수논거만.

| 조건 | 내용 | 미충족 시 |
|------|------|-----------|
| ①반증 가능 | 틀리는 증거를 사전 정의 | 가설이 아니라 신념 |
| ②구체성 | 검증 가능 수준 | 찾을 것 불명확 |
| ③검증 경로 | 검증 방법 명시 | 조사 후 판정 불가 |

**경쟁 가설 도출 (ACH, 필수):**
단일 가설만이면 확증편향. **경쟁 가설 2개 이상** (총 3개+).
- "이 의사결정에서 가능한 답이 3가지라면?"
- 상호배타적일 필요 없으나, 핵심 가설과 다른 결론 함의해야
- §2 HOW 증거-가설 매트릭스(ACH)에서 구조적 비교
- LIGHT: 경쟁 가설 1줄씩 (매트릭스 면제)

**충족도별 판정:**
- 3/3 + 경쟁가설 2개+ → 즉시 착수
- 3/3 + 경쟁가설 0-1개 → 경쟁 가설 보충 후 착수
- 2/3 → 조건부 진행: 미충족 명시 + §2 조사 중 보완
- 1/3 이하 → 재설계 (단, 형이 "일단 해봐"면 추정 보완 후 진행)

## 1-3. 프레임 설계

축 정의 + 전제 점검. 프레임 확정되어야 조사 방향 잡힘.

**절차:**
1. 축 정의 + 질문목록
2. **MECE 검증** — 체크리스트 통과
3. **자기반박 (DEEP, v2.5):** **틀밖** + **절대자** 발동 → 축 생성 자체 반박. "이 축들이 진짜 프레임인가? 빠진 축은? 잘못 박힌 축은?" 1문 자기반박
4. **전제 선언 (점검 아님):** P1/P2/P3 3가지 선언. 실질 점검은 §2-1 실시간 단일 집행처에서.
5. **폭 점검 1문 (FIG 전초):** "핵심·경쟁 가설이 각각 어떤 축에서 반증되는가?" 1문 선언. 축이 가설 반증 경로 미덮음 → 축 추가 or 가설 축소
6. 형 확인 → 승인 후 §2 진입

**MECE 체크리스트:**

| # | 점검 | 미통과 시 |
|---|------|-----------|
| ME-1 | 축 A와 축 B 조사 범위 겹침? → 통합 or 경계 재정의 | 축 간 오염 위험 |
| ME-2 | 모든 축 합치면 1-1에 답할 수 있나? → 못하면 누락 축 | 빈자리 발생 |
| ME-3 | 각 축이 독립적으로 기여? → 안 하면 불필요한 축 | 자원 낭비 |

**초점 설정:**
- 절대자(형 프레임) → 현황분석 → 축 × 렌즈
- 기본 렌즈: 고객·시장·경쟁·기술·규제·경제성
- 임의 프레임(SWOT/PESTLE/5Forces) 대체 금지 — 맥락에 맞는 축 설계
- **Issue Tree:** 주 질문(1-1)을 루트, 축을 L1 노드, 축별 하위 질문을 L2 노드. 3레벨 초과 = 축 분할

**[WHY 게이트]** 의사결정 연결점 + 가설 + 프레임 3개 채워짐 → §2
