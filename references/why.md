# §1 WHY — 왜 조사하는가

리서치의 존재 이유를 정의한다. 이 층위가 없으면 리서치가 자기목적화되어 "잘 조사했지만 아무 결정도 못 하는" 결과를 낳는다.

> [!info] LIGHT 모드
> LIGHT에서 §1 WHY는 **1줄 추정으로 대체**된다. 아래 절차는 DEEP 전용.
> - 1-1 의사결정 연결점: 1줄 추정 (매핑 생략)
> - 1-2 가설: 1줄 가설만 (3조건 점검 생략)
> - 1-3 프레임: 축은 DEEP과 동일하게 설정하되 전제 점검·형 승인 생략
> - WHY 게이트: 면제 — 바로 §2 진행
>
> LIGHT→DEEP 전환 시: LIGHT의 1줄 추정·가설·축이 DEEP WHY 1-1/1-2/1-3의 초안이 된다.

## 1-1. 의사결정 연결점

"이 리서치의 결론이 어떤 의사결정을 바꾸는가?"를 정의한다.

**절차:**
1. 의사결정 1-3개 명시 (형이 안 줬으면 추정 + 확신도 태깅)
2. 결론→선택지 매핑: "결론 A면 X를 선택, 결론 B면 Y를 선택"
3. 매핑 불가한 주제 = 스코프 밖 후보

**예시:**
```
의사결정: "이 시장에 진입할 것인가?"
  → 시장규모 ≥ X + 경쟁강도 ≤ Y → 진입
  → 그 외 → 보류 또는 피벗
```

## 1-2. 가설 수립 + 3조건

본격 리서치의 방향을 잡는 가설을 세운다. 가설이 없으면 조사가 산만해진다.

| 조건 | 내용 | 미충족 시 |
|------|------|-----------|
| ①반증 가능 | 틀리는 증거를 사전 정의 | 가설이 아니라 신념 |
| ②구체성 | 검증 가능한 수준으로 진술 | 무엇을 찾아야 하는지 불명확 |
| ③검증 경로 | 어떻게 검증할지 명시 | 조사 후에도 판정 불가 |

**경쟁 가설 도출 (ACH 원칙, 필수):**
단일 가설만으로 리서치하면 확증편향에 빠진다. 핵심 가설 외에 **경쟁 가설 2개 이상**(총 3개+)을 반드시 도출한다.
- "이 의사결정에서 가능한 답이 3가지라면 무엇인가?"로 시작
- 경쟁 가설은 상호배타적일 필요 없으나, 핵심 가설과 다른 결론을 함의해야 함
- §2 HOW에서 증거-가설 매트릭스로 경쟁 가설을 구조적으로 비교 (→ how.md §2-3 참조)
- LIGHT: 경쟁 가설 1줄씩만 (매트릭스 면제)

**충족도별 판정:**
- 3/3 + 경쟁가설 2개+ → 즉시 착수
- 3/3 + 경쟁가설 0-1개 → 경쟁 가설 보충 후 착수
- 2/3 → 조건부 진행: 미충족 조건 명시 + §2 조사 중 보완 의무
- 1/3 이하 → 가설 재설계 (단, 형이 "일단 해봐"면 추정 보완 후 진행)

## 1-3. 프레임 설계

축을 정의하고 전제를 점검한다. 프레임이 확정되어야 조사 방향이 잡힌다.

**절차:**
1. 축 정의 + 질문목록 작성
2. **MECE 검증 (필수):** 아래 체크리스트 통과 후 다음 단계 진행
3. 유효성 점검: 전제 3가지 명시 + "전제가 틀리면 프레임이 무너지는가?"
4. 형에게 확인 → 승인 후 §2 진입

**MECE 검증 체크리스트 (Issue Tree 원칙):**

| # | 점검 | 미통과 시 |
|---|------|-----------|
| ME-1 | 축 A와 축 B의 조사 범위가 겹치는가? → 겹치면 통합 또는 경계 재정의 | 축 간 오염 위험 |
| ME-2 | 모든 축의 조사 결과를 합치면 의사결정 연결점(1-1)에 답할 수 있는가? → 못하면 누락 축 존재 | 빈자리 발생 |
| ME-3 | 각 축이 의사결정에 독립적으로 기여하는가? → 안 하면 불필요한 축 | 자원 낭비 |

**초점 설정:**
- 절대자(형의 프레임) → 현황분석 → 축 × 렌즈
- 기본 렌즈: 고객, 시장, 경쟁, 기술, 규제, 경제성
- 축 적합성 1회 점검 (불필요한 축? 누락된 축?)
- 임의 프레임(SWOT/PESTLE/5Forces) 대체 금지 — 맥락에 맞는 축을 설계
- **Issue Tree 구조화:** 주 질문(1-1 의사결정)을 루트로, 축을 L1 노드로, 축별 하위 질문을 L2 노드로 배치. 계층이 3레벨을 초과하면 축 분할 검토

**[WHY 게이트]** 의사결정 연결점 + 가설 + 프레임 3개 채워짐 → §2 진입
