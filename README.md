# Research Frame

**Systematic research framework with LIGHT and DEEP modes.**

Structures any research task along three axes: WHY (purpose, hypotheses, framing), HOW (axis-by-axis investigation, bias prevention, cross-analysis), and SCOPE (boundaries, resources, termination criteria).

### Example Prompts

```
"Do a light research on X" → breadth-first scan, 1-2 sources per axis, minimal depth
"Deep research on Y" → full pipeline: frame hypotheses→axis research→cross-validate→converge
"Research this" → defaults to DEEP mode
```

### How It Works

1. **WHY** — Define the decision point, form 3 hypotheses, set the research frame
2. **HOW** — Independent axis-by-axis investigation → bias checks → cross-analysis
3. **SCOPE** — Set boundaries, resource limits, and termination criteria
