## §INV FINAL_RENDER_GATE (산출물 본질 보호 — 분리형)

| 항목 | 정의 |
|------|------|
| SCOPE | 사용자에게 전달되는 최종 산출물에만 적용 |
| INTERNAL | 내부 처리 라벨(축·층·Phase·모드 등)은 작업용으로 유지 OK |
| RULE | 외부 노출 시점 → 산출물·대화 = 인간 언어. 작업 라벨 ZERO. (1만 페이지 1단어 = FAIL) |
| 판정 | "이 단어, 이 대화 밖 사람이 사전 없이 읽을 수 있나?" NO → 작업 라벨 → 금지 |
| ALLOW | 업계 전문용어(MECE·KPI·LIGHT·DEEP·TURBO·EVIDENCE_BODY·SOURCE_CONTRAST·CLAIM_PROVENANCE) · 고유명사(출처명·기관명) · 법조문 |
| CONVERT | 라벨 발견 → 실명·평문 풀어쓰기. 예) 내부 처리 라벨(7층사고·5장르·시스템3·본질5)은 작업용으로 유지. 사용자 노출 _research/.md 산출물에서만 평문화 |
| SELF_CHECK | 산출물 출력 직전(내부 처리 단계는 제외)에서 자체 스캔. 1개라도 발견 = 차단·재작성 |
